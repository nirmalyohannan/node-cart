NodeCart API

Welcome to NodeCart API

A robust e-commerce backend API built with Node.js, Express, and MongoDB.

Features

  • User authentication and authorization
  • Product management
  • Shopping cart functionality
  • Order processing
  • RESTful API design

Getting Started

To get started with the API, you can make requests to the following endpoints:

GET /api/users - Get all users
POST /api/users/register - Register a new user
POST /api/users/login - User login

Installation

  1. Clone the repository
  2. Run npm install to install dependencies
  3. Create a .env file and set up your environment variables
  4. Run node index.mjs to start the server

Environment Variables

Create a .env file in the root directory with the following variables:

PORT=3000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
